# Legacy Servicing Provider Registry

## Purpose
Maps non‑NPI servicing provider identifiers (A‑prefix, M‑prefix, malformed IDs) to:
- Provider category  
- Claim volume  
- Service category patterns  

## Why It Exists
Many Medicaid servicing providers are atypical and do not have NPIs:
- Transportation vendors  
- Behavioral health agencies  
- County social service departments  
- Home‑ and community‑based service providers  

## Key Fields
- LEGACY_ID  
- PROVIDER_CATEGORY  
- CLAIM_VOLUME  
- SAMPLE_SERVICE_CATEGORY  

## Integration
Used by:
- Provider Integrity Scorecard  
- DQ Dashboard  
- Program Integrity analysis
